### Added

- New `pages/18_audio_chat_realtime.py` example: a two-way voice chat that streams microphone audio to OpenAI's Realtime API (`gpt-realtime`) over WebSocket and plays the model's spoken response back to the browser. Exercises the `sink_audio_track` + `source_audio_track` pair end-to-end (mic in → 24 kHz PCM → OpenAI → 24 kHz PCM → speaker) on independent clocks, with server-side VAD for turn-taking and barge-in.
